文件命名推荐（最被广泛识别）：
README.zh.md / README.zh-Hans.md（简中）
README.zh-TW.md / README.zh-Hant.md（繁中）
README.ja.md（日）
README.ko.md（韩）
README.ru.md（俄）

想让切换按钮更好看？可以去 https://shields.io 自己定制颜色、图标。
很多中国作者直接把 README.md 做成中文，把英文版叫 README.en.md，这样国内用户打开仓库默认就是中文。
